Behind the scenes

What algorithm is used?

Recommendation method

MoodStream uses a rule-based recommendation algorithm, not a trained machine learning model. Each movie has one or more mood tags such as happy, romantic, or intense.

How ranking works

When you select a mood, the app filters the catalog to movies tagged with that mood, then sorts the results by rating in descending order. If two movies are close, newer releases are preferred, and title order acts as the final tie-breaker.

Why it feels instant

Because the logic is deterministic and the catalog is curated, the recommendation appears immediately without network-heavy model inference. This makes the project simple to host and easy to explain.

The original desktop Tkinter experience is still in this repository. This web version was added so the project can be deployed on Vercel.
